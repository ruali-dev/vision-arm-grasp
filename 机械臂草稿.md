机械臂草稿

视觉抓取机械臂



基于视觉引导的机械臂抓取系统设计与实现 2024.12 -- 2025.01

Python / OpenCV / C++ / Arduino 个人项目(B站可演示:BV1uHrmYbEYr)



基于 3D 打印机械臂与 USB 相机构建视觉引导抓取系统，上位机运行于 Ubuntu,Arduino 负责舵机执行控制

使用 OpenCV 进行基于颜色的目标检测，通过像素到机械臂坐标的映射完成目标定位

将机械臂逆运动学核心算法以 C++ 实现并编译为动态库，由 Python 调用以提升运行效率

通过 JSON 串口协议下发关节角度指令，实现目标检测、抓取、投放与复位的完整流程



视频介绍

做了个视觉抓取机械臂，机体网上找的图纸，用3D打印出机身，用的N100工控机（Ubuntu22.04)+ArduinoUNO进行控制，视觉部分采用USB相机+opencv，运动学解算这部分参考了b站别的大佬的思路，用三角函数求解关节角，但是我把这部分放在上位机处理，然后用json发给arduino，程序主体使用python,然后运动学结算部分使用C++，编译成.so后放在python里面，这样保证了性能。可惜买的mg996舵机磨损了一个，就先展示抓一个吧。
